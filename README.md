# ftree

Utility for printing filesystem as a tree. (UNIX only for now :()

## Options

- `-a` - print all files
- `-d` - directory only
- `-s` - print the size of each file
- `-h` - print size of file in human readable format
- `-nc` - turn off colored output
- `-A` - use only ASCII characters in output
